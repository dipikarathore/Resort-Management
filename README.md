## âš™ï¸ Installation & Setup

Follow these steps to run the project locally:

1. **Install XAMPP / WAMP Server**

   * Download from: [https://www.apachefriends.org/](https://www.apachefriends.org/)

2. **Clone the Repository**

   ```bash
   git clone https://github.com/dipikarathore/Resort-Management-System.git
   ```

3. **Move Project to htdocs**

   ```
   C:/xampp/htdocs/Resort Management
   ```

4. **Create Database**

   * Open **phpMyAdmin**
   * Create a database named: `hotel`
   * Import the file: `hotel.sql`

5. **Run the Project**

   ```
   http://localhost/Resort%20Management/
   ```
